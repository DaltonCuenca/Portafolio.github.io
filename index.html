ChatBot sencillo con python

Realicé un chatbot utilizando un archivo JSON de patrones y respuestas. Toma una entrada del usuario y un diccionario de patrones y respuestas, y devuelve una respuesta aleatoria del diccionario si la entrada del usuario coincide con alguno de los patrones. GITHUB

Tres en raya con python

El juego comienza con un tablero vacío en el que los jugadores van marcando las casillas con su símbolo. El primer jugador en lograr una línea de tres símbolos iguales gana la partida. Si todas las casillas se llenan y no hay una línea de tres símbolos iguales, se declara un empate. GITHUB
